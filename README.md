# Real-Estate-Price-Analysis
Predicting housing prices using regression models and data analysis in Python.

## Project Overview
This project predicts housing prices using regression models and data analysis techniques in Python.

## Dataset
The dataset contains housing information such as:
- Square footage
- Number of bedrooms
- Number of floors
- Location factors

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Selection
4. Regression Modeling
5. Model Evaluation using R² score

## Files
- real_estate_price_prediction.ipynb

## Outcome
Built a predictive model to estimate housing prices based on property features.
